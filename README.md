# 🚀 CodeSage Lite – AI Code Analyzer

CodeSage Lite is a lightweight AI-powered code analysis tool that helps developers quickly understand the efficiency of their code.
It analyzes pasted code to detect **time complexity**, identify **redundant loops**, and provide **optimization suggestions**.

This project demonstrates the combination of **DSA concepts, backend development, and simple AI-based analysis**, making it a great portfolio project for students and aspiring developers.

---

# 📌 Features

✅ Detects approximate **time complexity** (O(1), O(n), O(n²), etc)
✅ Identifies **redundant loops** in the code
✅ Suggests **possible optimizations**
✅ Simple and clean **web interface**
✅ Beginner-friendly **AI code analysis logic**

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```
git clone https://github.com/yourusername/CodeSage_Lite.git

cd CodeSage_Lite
```

---

## 2️⃣ Install Backend Dependencies

Navigate to backend folder:

```
cd backend

pip install -r requirements.txt
```

---

## 3️⃣ Run Flask Server

```
python app.py
```

The backend server will start at:

```
http://127.0.0.1:5000
```

---

## 4️⃣ Run Frontend

Open the following file in your browser:

```
frontend/index.html
```

Now paste any code and click **Analyze**.

---

# 🧠 How It Works

1. User pastes code in the web interface.
2. The frontend sends the code to the Flask backend.
3. Backend analyzes the code using pattern detection.
4. The system:

   * Counts loops
   * Detects nested loops
   * Suggests optimizations
5. Results are displayed instantly.
